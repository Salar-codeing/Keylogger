What is this?
This is a simple python key logger that will write whatever a person types on his/her keyboard in a text file.
You can play around with the code and then make it to send the data to your email address.

How to use this?
You just need to run the code and while it is running it will write everything in a file called: “keylogs.txt” while you stop programming.
